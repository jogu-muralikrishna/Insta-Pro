# Insta-Pro

===================================
INSTAGRAM DOWNLOADER
===================================

A complete web application to download Instagram content including:
- Posts (photos & videos)
- Reels
- Stories
- Highlights
- Profile pictures

SETUP INSTRUCTIONS:
===================

1. Install Python 3.8 or higher

2. Install dependencies:
   pip install -r requirements.txt

3. Run the application:
   python run.py
   OR
   python app.py

4. Open your browser and go to:
   http://localhost:5000

FEATURES:
=========
- Download public content without login
- Login to download stories and user posts
- Beautiful web interface
- Auto cleanup of old files
- Zip file support
- Download history

IMPORTANT NOTES:
===============
- For private accounts, you need to login
- Don't use this for spam or violating Instagram's terms
- Your credentials are not stored permanently
- Respect content creators' rights

TROUBLESHOOTING:
===============
If you get login errors:
- Make sure your Instagram account is active
- Try logging in from browser first
- Some accounts may need 2FA

For download issues:
- Check your internet connection
- Make sure the URL/username is correct
- Public content must be accessible without login

SUPPORT:
=======
For issues, check console output for error messages
